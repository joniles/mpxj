Interface ProjectReader

  • All Known Implementing Classes:
    AbstractProjectReader, AstaDatabaseFileReader, AstaDatabaseReader, AstaFileReader, AstaTextFileReader, FastTrackReader, MerlinReader, MPDDatabaseReader, MPPReader, MPXReader, MSPDIReader, PhoenixReader, PlannerReader, PrimaveraDatabaseReader, PrimaveraPMFileReader, PrimaveraXERFileReader, ProjectLibreReader, UniversalProjectReader

    public interface ProjectReader
    This interface is implemented by all classes which can read project files of any type and generate an ProjectFile instance from the contents of the file.
